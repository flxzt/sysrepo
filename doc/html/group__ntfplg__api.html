typedef int(* srntf_replay_next) (const struct lys_module *mod, const struct timespec *start, const struct timespec *stop, struct lyd_node **notif, struct timespec *notif_ts, void *state)

Replay the next notification of a module.

Parameters
[in]modSpecific module.
[in]startNotifications with earlier timestamp cannot be returned.
[in]stopNotifications with equal or later timestamp cannot be returned.
[out]notifNotification data tree.
[out]notif_tsNotification timestamp.
[in,out]stateArbitrary state to keep track of returned notifications, is NULL on first call.
Returns
SR_ERR_OK on success;
SR_ERR_NOT_FOUND if there are no more notifications, state was freed.
Sysrepo error value on error, state was freed.
